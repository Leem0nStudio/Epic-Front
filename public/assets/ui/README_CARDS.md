# Card Images for Enemies

Place the following card images in this folder:

- `card_zombie.png` - Card image for Zombie enemy
- `card_goblin.png` - Card image for Goblin enemy  
- `card_baphomet.png` - Card image for Baphomet enemy

These images will be displayed when players tap on the enemy icons.

Naming convention: `card_{enemy_name}.png`
