# UI Assets

Elementos de interfaz para la aplicación.

## Archivos esperados

- `button_*.png` - Botones UI
- `icon_*.png` - Iconos personalizados
- `frame_*.png` - Marcos y bordes
- `panel_*.png` - Paneles de interfaz

## Especificaciones

- Formato: PNG con transparencia
- Resolución: 1-2x (escalable)
- Color: Según paleta del juego
