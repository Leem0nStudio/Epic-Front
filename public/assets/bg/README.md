# Background Assets

Fondos y capas para la pantalla home y efectos parallax.

## Archivos esperados

- `home_bg.png` - Fondo principal de home
- `battle_bg_*.png` - Fondos de batalla (diferentes escenarios)
- `parallax_layer_1.png` - Primera capa de parallax (lejana)
- `parallax_layer_2.png` - Segunda capa de parallax
- `parallax_layer_3.png` - Tercera capa de parallax (cercana)

## Especificaciones

- Formato: PNG
- Resolución: Full HD (1920x1080) o múltiplos
- Capas parallax: Diferentes velocidades de scroll
- Contiene: Texturas, decoración, elementos del escenario
